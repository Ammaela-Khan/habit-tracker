Architecture and Deisgn: 
This app uses vanilla JavaScript, HTML, and CSS. 
1. index.html: focuses on UI layout and structure of the application.
2. styles.css: Styling and color variables.
3. app.js: Main brain and logic for habits, rendering, event holding, and localStorage.
4. User interaction include adding, ticking, and deleting habits. 

On Windows:

2. Open folder
3. Double-click index.html (it will open is the browser)

On MacOS: 
1. Download or clone the repository
2. In ifnder, open the folder
3. Right-click index.html-> Open with -> Safari/Chrome
